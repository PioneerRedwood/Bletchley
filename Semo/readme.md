# git 사용법 복습

### 2021.03.12

### git 상태확인

- git status

  

#### git branch

- git branch => branch 확인
- git checkout 브랜치명 => branch 변경



### 파일 add, commit, push

- git add 파일
- git commit -m "설명"
- git push origin 브랜치명



### 파일 pull

- git pull origin 가져올 브랜치명 => 현재 설정 branch에 추가됨



### 디렉토리 삭제

- git rm -r 디렉토리명
